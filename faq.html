Troubleshoot video playback issues

+
If the video doesn’t start or playback is interrupted:
1. Check your internet connection.
2. Close applications and other browser tabs and windows that you are not using.
3. Try a different browser.
4. Try a different device if possible. If you are using a tablet or phone, try a PC or laptop.
5. Try playing another video from the course to see if there is a playback issueм.
6. Update your drivers if possible.
7. Open the video in a browser using an Incognito mode.
